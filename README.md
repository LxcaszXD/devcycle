O **DevCycle** é uma aplicação web focada inteiramente no ecossistema Front-end. O projeto foi construído com uma forte ênfase em organização de código, escalabilidade de estilos e, principalmente, na experiência do usuário (UX). Através da união de layouts responsivos e animações fluidas, o projeto demonstra boas práticas modernas de estruturação de interfaces web.

🌐 **Projeto em Produção:** [https://devcycle.lucaszdev.me](https://devcycle.lucaszdev.me)  

---

## 🎯 Destaques do Projeto

* **Design Responsivo:** Layout fluido e adaptável a diferentes tamanhos de tela, garantindo acessibilidade em dispositivos móveis e desktops.
* **Animações Avançadas:** Utilização do GSAP (GreenSock) para criar interações e transições de alto desempenho, enriquecendo a experiência visual.
* **Estilização Modular:** Uso de SCSS para manter o código CSS limpo, reutilizável e fácil de manter.
* **Componentização Visual:** Estruturação ágil utilizando o sistema de grid e componentes do Bootstrap.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estrutura semântica e acessível.
- **SCSS (Sass):** Pré-processador CSS para variáveis, aninhamento e modularização do layout.
- **JavaScript:** Controle da lógica de interface e manipulação do DOM.
- **[GSAP (GreenSock)](https://gsap.com/):** Biblioteca profissional para animações web de alta complexidade e performance.
- **[BoxIcons](https://boxicons.com/):** Biblioteca de ícones vetorizados para complementar o design da interface.

## 🚀 Como Executar o Projeto Localmente

Como este é um projeto exclusivamente Front-end (sem dependências de servidor ou banco de dados), executá-lo é muito simples e rápido.

### Passo a Passo

**1. Clone o repositório para a sua máquina:**
```bash
git clone https://github.com/LxcaszXD/devcycle.git
```

**2. Acesse a pasta do projeto:**
```bash
cd devcycle
```

**3. Abra no Navegador:**
- A forma mais simples é apenas dar um duplo clique no arquivo `index.html` para abri-lo no seu navegador padrão.
- **Recomendado:** Se você usa o VS Code, instale a extensão **Live Server**. Clique com o botão direito no arquivo `index.html` e selecione `Open with Live Server` para ver as atualizações em tempo real enquanto edita o código.
